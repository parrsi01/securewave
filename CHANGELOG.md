# Changelog

All notable changes to SecureWave VPN will be documented in this file.

## [1.0.0] - 2026-01-25

### UI Version 1.0.0 - True v1.0 Fresh Start

This is the first official release. All previous UI versions have been superseded.

### Brand Identity
- New logo: Simple shield with checkmark (universal "secure" symbol)
- Primary color: Teal (#0D9488 to #14B8A6)
- Design philosophy: Calm, trustworthy, beginner-friendly

### Design System
- Complete DESIGN_SYSTEM.md documentation
- Teal color palette with rationale
- WCAG 2.1 AA accessibility compliance
- Large touch targets (52px minimum)
- Generous spacing for readability

### Website
- Clean, calm light theme (default)
- Dark theme with toggle
- Mobile-first responsive design
- Simple shield logo at all sizes
- Beginner-friendly copy and layouts

### Flutter App
- Teal theme matching website
- Material 3 design
- Large button sizes for touch
- Consistent color system

### Files Changed
- `/static/img/logo.svg` - New teal shield
- `/static/img/logo-dark.svg` - Dark background variant
- `/static/img/logo-mark.svg` - App icon
- `/static/favicon.svg` - Browser favicon
- `/static/css/professional.css` - Complete rewrite
- `/securewave_app/lib/core/theme/app_theme.dart` - Flutter theme
