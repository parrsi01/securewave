Quick Troubleshooting

1
Refresh the page

Press Ctrl+R (or Cmd+R on Mac) to reload

2
Clear browser cache

Press Ctrl+Shift+Delete and clear cached data

3
Check application status

If running locally, ensure server is running with bash deploy.sh local

4
Verify dependencies

Run bash test-app.sh to check if all dependencies are installed
